
## Питон для самых маленьких


### Почему Python?

Python — один из самых дружелюбных языков программирования. Его создали так, чтобы код читался почти как обычный английский текст. Именно поэтому Python часто выбирают для первого знакомства с программированием.

### 1. Переменные: наши «ячейки памяти»

Представьте шкафчик с ячейками, куда можно положить разные вещи. В программировании такие «ячейки» называются **переменными**.

**Что важно знать:**

-   У каждой переменной есть **имя** (как наклейка на шкафчике).
    
-   В ней хранится **значение** (то, что лежит внутри).
    
-   Python сам «угадывает» тип данных (число, текст и т. д.).
    

**Примеры:**

-   `возраст = 15` — как табличка с возрастом на двери комнаты.
    
-   `город = "Рязань"` — как адрес на конверте.
    
-   `есть_паспорт = True` — как ответ «да/нет» на вопрос.
    

```
# Записываем данные
имя = "Алексей"
рост = 1.78
класс = 9

# Выводим на экран
print(имя, рост, класс)

```

> **Совет:** выбирайте понятные имена переменных (не `x`, а `возраст`), чтобы не запутаться в дальнейшем.

### 2. Ввод и вывод: диалог с компьютером

Компьютер может **спрашивать** и **отвечать**. Для этого есть две волшебные команды:

-   `print()` — «говорит» (выводит текст на экран).
    
-   `input()` — «слушает» (ждёт ввода с клавиатуры).
    

**Пример диалога:**

```
print("Привет! Как тебя зовут?")
имя = input()  # Ждём, пока пользователь введёт имя
print("Рад познакомиться, " + имя + "!")

```

**Тонкости:**

-   `input()` всегда возвращает **строку** (текст).
    
-   Чтобы получить число, нужно преобразовать: `int(input())`.
    

```
print("Сколько тебе лет?")
возраст = int(input())  # Превращаем текст в число
print("Через год тебе будет", возраст + 1)

```

### 3. Арифметика

Python умеет выполнять все привычные математические операции:

-   `+` — сложение (как сложить яблоки в корзину).
    
-   `-` — вычитание (как отдать часть конфет).
    
-   `*` — умножение (как посчитать площадь комнаты).
    
-   `/` — деление (как поделить пиццу на куски).
    

**Неожиданные операции:**

-   `//` — целочисленное деление (сколько целых кусков можно отрезать).
    
-   `%` — остаток от деления (что останется «лишним»).
    
-   `**` — возведение в степень (как умножить число само на себя несколько раз).
    

```
a = 17
b = 5

print(a // b)  # 3 (можно отрезать 3 целых куска)
print(a % b)   # 2 (остаток 2)
print(2 ** 3)  # 8 (2 × 2 × 2)

```

### 4. Условия: если… то…

Жизнь полна выборов: «Если пойдёт дождь — возьму зонтик». В программировании это реализуется через `if`.

**Базовый шаблон:**

```
if условие:
    # что делать, если условие верно
else:
    # что делать, если условие неверно

```

**Пример с погодой:**

```
температура = 12

if температура > 20:
    print("Можно идти в шортах!")
else:
    print("Лучше надеть куртку.")

```

**Сложные условия:**

-   `and` — «и» (должны выполняться оба условия).
    
-   `or` — «или» (достаточно одного условия).
    
-   `not` — «не» (обратное условие).
    

```
дождь = False
ветер = True

if not дождь and not ветер:
    print("Отличная погода для прогулки!")

```

### 5. Циклы

Представьте, что нужно написать 100 раз «Я люблю Python!». Вручную — долго, а с циклом — легко.

**Цикл `for`**: перебирает элементы (как листать страницы книги).

```
фрукты = ["яблоко", "банан", "апельсин"]

for фрукт in фрукты:
    print("Мне нравится", фрукт)

```

**Цикл `while`**: делает, пока условие верно (как ждать автобуса).

```
счётчик = 0

while счётчик < 5:
    print("Счётчик:", счётчик)
    счётчик = счётчик + 1  # увеличиваем на 1

```

> **Важно:** в `while` нужно следить, чтобы цикл когда‑нибудь закончился, иначе программа будет исполнятся бесконечно, пока не будет завершена через диспетчер задач.

### 6. Списки

Список — это как список покупок: несколько элементов в одном месте.

```
покупки = ["хлеб", "молоко", "сыр"]

print(покупки[0])  # "хлеб" (первый элемент, индекс 0)
покупки.append("яйца")  # добавили в конец
print(len(покупки))  # 4 (сколько всего)

```

**Заметки:**

-   Индексы начинаются с **0** (первый элемент — `[0]`).
    
-   `len()` — считает количество элементов.
    
-   `append()` — добавляет в конец.
    

### 7. Строки: работаем с текстом

Строки — это любой текст в кавычках. С ними можно:

-   Соединять: `"Привет" + "мир"` → `"Приветмир"`.
    
-   Умножать: `"Ха" * 3` → `"ХаХаХа"`.
    
-   Узнавать длину: `len("кот")` → `3`.
    

```
имя = "Мария"
приветствие = "Привет, " + имя + "!"
print(приветствие)  # "Привет, Мария!"

```

### 8. Функции

Функция — это как рецепт: описали один раз, а потом используем многократно.

**Как создать:**

```
def имя_функции(аргументы):
    # тело функции
    return результат

```

**Пример с приветствием:**

```
def поздороваться(имя):
    print("Привет, " + имя + "!")

# Используем
поздороваться("Анна")
поздороваться("Иван")

```

**Почему это удобно:**

-   Не нужно писать одно и то же много раз.
    
-   Код становится понятнее.
    
-   Можно «настраивать» функцию через аргументы.

> **Важно!** Функция может не возвращать значение! Тогда она превращается в *метод*!
    

### 9. Комментарии

Комментарии — это пояснения, которые видит только программист (компьютер их игнорирует).

```
# Это комментарий — пояснение к коду
x = 10  # Можно писать и тут

"""
Также комментарии можно писать в виде 
многострочного текста - блока, выделенного тройными кавычками!
"""

```

**Когда использовать:**

-   Объяснять сложные участки.
    
-   Напоминать себе о планах.
    
-   Делать код понятнее для других.
    

### 10. Отступы: «грамматика» Python

В Python отступы — не просто красота, а **обязательная часть синтаксиса**. Они показывают, где начинается и заканчивается блок кода.

**Правильно:**

```
if x > 5:
    print("x больше 5")  # Внутри if
    print("Это тоже внутри")
else:
    print("x меньше или равно 5")

```

**Ошибочно:**

```
if x > 5:
print("x больше 5")  # Ошибка: нет отступа!

```

> **Правило:** все строки в одном блоке должны иметь одинаковый отступ (обычно 4 пробела или Tab). В программе нельзя комбинировать пробелы и Tab.
